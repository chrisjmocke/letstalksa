Deployment flow

  1. Initialize git and push to a new GitHub repository.
  2. Enable Pages using GitHub Actions in repository settings.
  3. Keep the CNAME file set to web.limpopovoice.co.za.
  4. Create DNS CNAME: web -> your GitHub Pages host target.
  5. Verify HTTPS provisioning and test from multiple devices.
